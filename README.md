# MRP (Material Requirements Planning) Module for FrontAccounting

Advanced material requirements planning for manufacturing operations.

## Features

- **Demand Forecasting**: Sales forecast and demand planning
- **MRP Calculations**: Automatic calculation of material requirements
- **Production Planning**: Master production scheduling
- **Inventory Optimization**: Safety stock and reorder point calculations
- **Supplier Integration**: Integration with supplier lead times and capacities
- **Capacity Planning**: Production capacity analysis and planning
- **What-if Analysis**: Scenario planning and analysis

## Requirements

- FrontAccounting 2.4+
- PHP 8.0+
- Manufacturing environment
